## 🖱️ 基本操作

### 鼠标操作

## 💕 互动功能

通过右键菜单可以与小白进行各种互动：

### 贴贴互动

| 功能 | 效果 | 消耗 |
|------|------|------|
| 🤗 **贴贴** | 增加快乐值 | 能量值- |
| 👋 **拍一拍** | 短暂互动 | 无 |
| 💪 **锻炼** | 增加快乐 | 能量值-- |
| 🔋 **充电** | 增加快乐+能量 | 无 |
| 🍖 **投喂小白** | 快乐+能量 | 仅在饥饿时有效 |
| 😘 **吧唧** | 增加快乐 | 无 |
| 🪶 **鸡毛丸子** | 增加快乐 | 能量值-- |
| 🚶 **遛小白** | 增加快乐 | 能量值-- |
| 🛏️ **睡觉** | 进入睡眠状态 | 无 |

### 互动效果说明

- **快乐值**：代表小白的开心程度
- **能量值**：代表小白的体力状态
- 当两项都为 **0** 时，小白会死亡 💀
- 死亡后约 **30分钟** 会自动复活 ✨

## 🎮 趣味游戏

小白内置了多款经典小游戏，点击游戏图标即可开始！

### 游戏列表

#### 1. 🎯 数独 Sudoku
- **难度**：简单 / 中等 / 困难
- **操作**：选择格子 → 点击数字填入
- **规则**：每行、每列、每宫格1-9不重复
- **功能**：
  - 🗑️ **删除**：擦除填入的数字
  - 💡 **提示**：显示正确数字（最多10次）
  - 📝 **解答**：显示完整答案
  - ⌨️ 数字键 1-9 直接填入
  - ⌫ Backspace 删除

#### 2. 🎲 华容道 Klotski
- **目标**：将曹操（红色方块）移到最下方中央
- **操作**：点击选中棋子 → 点击空格移动
- **规则**：只能移动到相邻空格
- **胜利**：曹操到达底部中央位置

#### 3. 🐍 贪吃蛇 Snake
- **操作**：方向键控制蛇的移动方向
- **规则**：吃食物增长蛇身，碰到墙壁或自身游戏结束
- **目标**：尽可能获得更高分数

#### 4. ⬛ 俄罗斯方块 Tetris
- **操作**：← → 移动，↑ 旋转，↓ 加速下落
- **规则**：消除完整行获得分数
- **目标**：挑战更高分数

#### 5. 🔢 2048
- **操作**：方向键移动所有数字方块
- **规则**：相同数字碰撞会合并成两数之和
- **目标**：合成 2048！

#### 6. ⚔️ 五子棋 Gomoku
- **操作**：点击棋盘放置棋子
- **规则**：先连成5子获胜
- **模式**：人机对战 / 双人对战

#### 7. ⭕ 井字棋 TicTacToe
- **操作**：点击放置 X 或 O
- **规则**：三连获胜
- **模式**：人机对战 / 双人对战

#### 8. 💣 扫雷 Minesweeper
- **操作**：左键点击翻开格子，右键标记
- **规则**：避开所有地雷
- **目标**：找出所有安全格子

#### 9. 🧱 推箱子 Sokoban
- **操作**：方向键移动角色
- **规则**：将所有箱子推到目标位置
- **目标**：用最少步数完成关卡

#### 10. 🐹 仓鼠养殖 Hamster
- **玩法**：养可爱的小仓鼠
- **功能**：喂食、玩耍、观察成长

#### 11. 🏓 乒乓球 Pong
- **操作**：↑ ↓ 控制球拍
- **规则**：乒乓球游戏
- **目标**：击败对手

#### 12. 🎯 打地鼠 Whackamole
- **操作**：点击冒出的地鼠
- **规则**：地鼠出现后快速点击得分
- **目标**：在限定时间内获得最高分

#### 13. 🚢 海战棋 Battleship
- **玩法**：经典的猜测舰队位置游戏
- **规则**：在网格上标记攻击位置
- **目标**：击沉所有敌方舰船

#### 14. 🎴 小型游戏合集
- **俄罗斯方块** / **贪吃蛇** / **消消乐** 等经典游戏

#### 15. 🎮 小白对战游戏
- **坦克大战**：控制坦克战斗
- **更多游戏持续更新中...**

## ❓ 常见问题

### Q1: 小白为什么会消失？
**A**: 当能量值和快乐值都为0时，小白会进入休眠状态。约30分钟后会自动复活。

### Q2: 如何让小白保持活力？
**A**: 定期与小白互动（贴贴、充电、投喂），保持快乐值和能量值在50以上。

### Q3: 如何隐藏状态栏？
**A**: 右键菜单 → 「显示/隐藏状态栏」

### Q4: 小白不听话怎么办？
**A**: 重启程序或删除配置文件（位于 `%LOCALAPPDATA%\DesktopPet\config.json`）

### Q5: 如何调整小白大小？
**A**: 右键 → 「配置」 → 「外观设置」 → 调整大小滑块

### Q6: 游戏无法打开？
**A**: 确保已安装所有依赖库，如 PyQt5、Pillow 等
