## 方法三：分步执行

### 步骤1：检查依赖
```cmd
pip list | findstr /i "pyinstaller pyqt5 pillow"
```

如果依赖缺失：
```cmd
pip install pyinstaller PyQt5 Pillow win10toast plyer
```

### 步骤2：PyInstaller打包
```cmd
cd "小白-源代码"
pyinstaller 小白.spec
```

### 步骤3：Inno Setup创建安装程序
安装 Inno Setup 6 后，双击 `installer.iss` 文件
